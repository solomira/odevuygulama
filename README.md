1. PROJE GENEL BAKIŞ
--------------------------------------------------------------------------------

Bu uygulama, React Native ve Expo framework'ü kullanılarak geliştirilmiş bir 
mobil not hesaplama uygulamasıdır. Öğrencilerin vize ve final notlarını girerek 
dönem notlarını hesaplamasına, harf notlarını görmesine ve hedef not belirlemesine 
olanak tanır.

Kullanılan Teknolojiler:
- React Native 0.72.6    : Cross-platform mobil uygulama geliştirme
- Expo 49                : React Native geliştirme araç seti
- TypeScript             : Tip güvenli JavaScript
- React Navigation 6     : Sayfa yönlendirme sistemi
- AsyncStorage           : Yerel veri depolama
- Context API            : Global state yönetimi
--------------------------------------------------------------------------------

Bu uygulama, modern React Native geliştirme pratiklerini kullanarak 
tasarlanmış, kullanıcı dostu bir not hesaplama aracıdır. 

Öne Çıkan Özellikler:
- TypeScript ile tip güvenliği
- Context API ile state yönetimi
- Modüler bileşen yapısı
- Çoklu dil desteği (Türkçe/İngilizce)
- Karanlık/Aydınlık tema
- Özelleştirilebilir not ağırlıkları
- Harf notu sistemi
- Hedef not hesaplama
- Kalıcı veri saklama
- uygulama tarafımca Fiziksel cihaz ve Android Emulator ile test edilmiştir.
